XE – Currency Exchange Rate Lookup

Website: https://www.xe.com
Task: Look up currency exchange rates

Parameters:
  - from_currency: source currency code (e.g., "USD")
  - to_currency: target currency code (e.g., "EUR")

Fields to extract:
  - from_currency: source currency code
  - to_currency: target currency code
  - current_rate: current exchange rate
  - inverse_rate: inverse exchange rate (1 target = X source)
  - high_30d: 30-day high rate
  - low_30d: 30-day low rate
  - avg_30d: 30-day average rate
  - volatility_30d: 30-day volatility
