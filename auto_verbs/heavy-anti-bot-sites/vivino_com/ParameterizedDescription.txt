Vivino – Wine Search

Search Vivino for wines by keyword.

Parameters:
- search_query (str): The search query (e.g. "Cabernet Sauvignon"). Default: "Cabernet Sauvignon"
- max_results (int): Maximum number of results to return. Default: 5

Returns:
- wines: list of WineResult
  - name (str): Wine name
  - winery (str): Winery name
  - region (str): Wine region
  - rating (str): Rating
  - num_ratings (str): Number of ratings
  - price (str): Price
