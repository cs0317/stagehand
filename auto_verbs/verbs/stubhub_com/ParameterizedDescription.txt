stubhub_search(query, max_results=5)

Search for event tickets on StubHub.
Navigate to the search results page for the given query.
Extract up to `max_results` events with event name, venue, date, and lowest ticket price.
Print the list of results.

Parameters:
  query       – search term, e.g. "NBA"
  max_results – maximum number of results to return (default 5)

URL pattern: https://www.stubhub.com/secure/search?q={query}
