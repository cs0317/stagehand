Care.com – Babysitter Search

Search for babysitters in a given location on Care.com and extract caregiver profiles.

Parameters:
  - location (str, default "Austin, TX"): City and state to search for babysitters.
  - max_results (int, default 5): Maximum number of caregiver profiles to extract.

Returns a list of caregiver profiles, each containing:
  - name (str): Name of the caregiver.
  - years_of_experience (str): Years of experience (e.g., "5 yrs"), or "N/A" if not shown.
  - hourly_rate (str): Hourly rate (e.g., "$15-20/hr"), or "N/A" if not shown.
  - rating_or_reviews (str): Rating or number of reviews (e.g., "4.8 stars", "12 reviews"), or "N/A" if not shown.
