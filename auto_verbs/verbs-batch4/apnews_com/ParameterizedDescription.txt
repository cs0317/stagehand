AP News – Search Articles

Search for news stories on apnews.com matching a given query and extract article details.

Parameters:
  - query (str, default "space exploration"): Search query string.
  - max_results (int, default 5): Maximum number of articles to extract.

Returns a list of articles, each containing:
  - headline (str): Article headline.
  - publication_date (str): Publication date, or "N/A" if not found.
  - summary (str): Summary or first paragraph of the article, or "N/A" if not found.
