Canva.com – Design Template Search

Search for design templates on Canva matching a given query and extract template details.

Parameters:
  - search_query (str, default "business presentation"): The search term for finding templates.
  - max_results (int, default 5): Maximum number of templates to extract.

Returns a list of templates, each containing:
  - template_name (str): Name of the template.
  - category (str): Category or type of the template (e.g., "Presentation", "Infographic"), or "N/A" if not shown.
  - dimensions (str): Dimensions or format of the template (e.g., "1920 × 1080 px", "16:9"), or "N/A" if not shown.
  - is_free (str): Whether the template is free or requires Pro (e.g., "Free", "Pro"), or "N/A" if not shown.
