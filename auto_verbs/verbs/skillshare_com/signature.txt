skillshare_search(query: str, max_results: int = 5) -> list[dict]

@dataclass
SkillshareClass:
    title: str      # Class title, e.g. "Intro to Illustrative Lettering"
    teacher: str    # Teacher name, e.g. "Chris Piascik"
    duration: str   # Class duration, e.g. "1h 25m" or "59m"
    students: str   # Number of students, e.g. "3.8k" or "425"

Input:
    query       – search keyword, e.g. "illustration"
    max_results – maximum number of classes to return (default 5)

Output:
    List of SkillshareClass dicts with keys: title, teacher, duration, students

URL pattern: https://www.skillshare.com/en/search?query={query}
Note: Page may crash intermittently; script includes retry logic
