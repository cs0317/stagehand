DeviantArt.com – Search Artwork

Search DeviantArt for artwork matching a given query and extract artwork listings.

Parameters:
  - search_query (str, default "digital fantasy art"): Search term for finding artwork.
  - max_results (int, default 5): Maximum number of artworks to extract.

Returns a list of artworks, each containing:
  - title (str): Title of the artwork.
  - artist_name (str): Name of the artist who created the artwork.
  - favorites (str): Number of favorites/likes the artwork has received, or "N/A" if not shown.
  - category (str): Category or tag of the artwork, or "N/A" if not shown.
