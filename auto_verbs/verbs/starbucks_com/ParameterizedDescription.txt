starbucks_store_locator(location, max_results=5)

Use the Starbucks store locator to find stores near a location.
Type the location into the search field and press Enter.
Extract up to `max_results` stores with name, address, hours, distance, and order type.
Print the list of results.

Parameters:
  location    – search location, e.g. "Manhattan, NY"
  max_results – maximum number of stores to return (default 5)

URL: https://www.starbucks.com/store-locator
Interaction: type location into search input, press Enter, parse body text
