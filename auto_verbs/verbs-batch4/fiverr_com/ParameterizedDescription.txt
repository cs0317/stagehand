Fiverr – Search Gigs

Search for freelance services on fiverr.com matching a given query and extract gig details.

Parameters:
  - query (str, default "logo design"): Search query string.
  - max_results (int, default 5): Maximum number of gigs to extract.

Returns a list of gigs, each containing:
  - seller_name (str): Name of the seller.
  - service_title (str): Title of the gig/service.
  - starting_price (str): Starting price, or "N/A" if not found.
  - rating (str): Seller rating, or "N/A" if not found.
  - num_reviews (str): Number of reviews, or "N/A" if not found.
