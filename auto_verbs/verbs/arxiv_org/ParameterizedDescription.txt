Arxiv.org – Search Research Papers

Parameters:
  - query (str): The search query for papers (e.g. "transformer architecture")
  - max_results (int): Maximum number of papers to return (e.g. 5)

Steps:
  1. Navigate to https://arxiv.org
  2. Search for papers matching {query}
  3. Extract up to {max_results} papers, each with:
     - title
     - authors
     - abstract snippet
     - submission date
  4. Return/print the list of papers
